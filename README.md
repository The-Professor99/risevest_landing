# Risevest Landing: Frontend Test

## Run Locally

Clone the project

```bash
  git clone https://github.com/The-Professor99/risevest_landing.git
```

Go to the project directory

```bash
  cd risevest_landing
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
